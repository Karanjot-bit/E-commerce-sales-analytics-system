# 🛒 E-Commerce Sales Analytics System

## 📌 Project Overview

This project is an E-Commerce Sales Analytics System developed using Python and SQL. The system analyzes sales data to generate meaningful insights such as customer behavior, product performance, and sales trends.

The main objective is to transform raw data into useful information that can help businesses make better decisions.

---

## 🎯 Objectives

* Clean and preprocess raw e-commerce data
* Store data in a structured database (MySQL)
* Perform data analysis using SQL queries
* Visualize trends using Python
* Generate business insights

---

## 🧰 Technologies Used

* Python
* MySQL
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

---

## 📊 Dataset

The dataset contains:

* Customer details
* Product information
* Order details
* Transaction data

(Source: Kaggle)

## ⚙️ Project Workflow
